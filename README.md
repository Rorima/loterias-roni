# GDD resumido

Objetivo:

Acumular a maior quantidade possível de pontos.

Existem cinco áreas do jogo.

🏠 Início

Botão de clique
Cada clique vale exatamente +1 ponto
Nunca haverá upgrades para aumentar o valor do clique.

🏦 Renda Fixa

O jogador pode investir pontos.
O dinheiro investido fica bloqueado.
Enquanto o aplicativo estiver aberto, gera dividendos lentamente.
O rendimento deve ser facilmente configurável por constantes.
O jogador pode resgatar tudo a qualquer momento.

🎰 Apostas

Multiplicadores:

2x
3x
5x
10x
100x

As probabilidades ficarão em constantes para facilitar alteração futura.

📈 Bolsa

Sempre cinco ativos.

Cada ativo possui:

nome
cor
tendência
volatilidade
índice (0-100)
gráfico

Atualiza a cada 10 segundos.

Quando chega a zero:

ativo é removido
novo ativo é criado

📊 Estatísticas

Registrar:

total de cliques
total apostado
total ganho
total perdido
maior aposta
maior vitória
maior derrota
patrimônio máximo
lucro da bolsa
lucro da renda fixa
tempo jogado
